# 🌌 Spheres of Power — Трехмерные Шахматы (3D Chess Engine)

[![Python Version](https://shields.io)](https://python.org)
[![Engine](https://shields.io)](https://ursinaengine.org)
[![License: MIT](https://shields.io)](https://opensource.org)

**«Сферы Власти» (Spheres of Power)** — это концептуальная настольная стратегия, переносящая классические шахматные механики в полноценное трехмерное пространство (кубическая матрица $8 \times 8 \times 8$). Игра симулирует космическое или футуристическое сражение, где тактические маневры происходят не только на плоскости, но и по вертикальным осям, а также сквозь межпространственные диагонали.

---

## 🚀 Основные Особенности

*   **Полноценное 3D-пространство:** Поле боя состоит из 8 вертикальных слоев по 64 клетки в каждом (всего 512 игровых ячеек). Координаты описываются вектором `[X, Y, Z]`.
*   **Эшелонированная оборона:** Увеличенный стартовый стек фигур (26 штук у каждого игрока вместо 16) распределен по двум базовым уровням для предотвращения быстрой вязкости геймплея.
*   **Интерактивная камера:** Свободное вращение вокруг кубического каркаса, приближение и панорамирование в реальном времени.
*   **Встроенный автоматический QA-модуль:** Система юнит-тестов проверяет математическую точность генерации координат и логику ходов перед каждым запуском графической оболочки.

---

## 🎮 Правила Игры и Функции Фигур

Положение любой фигуры задается тремя значениями: `X` (горизонталь), `Y` (вертикаль на плоскости), `Z` (номер слоя от 0 до 7).

### Фигуры и их 3D-механики:
*   **Пешка (Дрон):** Ходит на 1 клетку вперед по оси Y или поднимается на 1 слой по оси Z (без атаки). Атакует по трехмерной диагонали (вперед-вверх/вперед-вниз).
*   **Ладья (Генератор поля):** Прошивает куб насквозь по одной выбранной оси ($X$, $Y$ или $Z$). Идеальна для блокирования целых вертикальных секторов.
*   **Слон (Перехватчик):** Перемещается по трехмерным диагоналям (изменение всех трех координат должно быть эквивалентным: $\Delta X = \Delta Y = \Delta Z$).
*   **Конь (Прыгун подпространства):** Игнорирует любые препятствия. Его ход — это трехмерный вектор $2+1$ (например, 2 слоя вверх по Z и 1 клетка вперед по Y).
*   **Ферзь (Матриарх):** Абсолютное оружие. Объединяет векторы Ладьи и Слона во всех направлениях куба.
*   **Король (Ядро системы):** Перемещается на 1 клетку в любом направлении на текущем слое или на смежные уровни по вертикали/диагонали. Цель игры — поставить **Матричный мат** (блокировка на текущем и соседних слоях).

## 🛠 Установка и Запуск

### Требования
Для работы проекта необходим установленный интерпретатор **Python версии 3.8 или выше**.

### 1. Клонирование репозитория
```bash
git clone https://github.com
cd spheres-of-power
```

### 2. Установка зависимостей
Проект использует легковесный игровой движок **Ursina**, построенный на базе Panda3D. Установите его с помощью pip:
```bash
pip install -r requirements.txt
# или напрямую:
pip install ursina
```

### 3. Запуск игры
```bash
python spheres_of_power.py
```
*При запуске приложение автоматически прогонит юнит-тесты в консоли перед инициализацией 3D-окна.*

## 🕹 Управление в игре

*   **Вращение камеры:** Зажмите **Правую Кнопку Мыши (ПКМ)** и двигайте мышь, чтобы осмотреть куб под любым углом.
*   **Панорамирование (сдвиг камеры):** Зажмите **Колесико мыши (СКМ)** и перемещайте мышь для сдвига центра обзора.
*   **Масштабирование:** Прокручивайте колесико мыши вперед/назад для приближения или отдаления.
*   **Выбор фигуры:** Нажмите **Левую Кнопку Мыши (ЛКМ)** по фигуре. Выбранная фигура подсветится желтым цветом.
*   **Совершение хода:** После выбора фигуры нажмите **ЛКМ** по целевой клетке (пустой или с фигурой противника). Если ход невалиден, система заблокирует его и выведет предупреждение в лог.

## 📊 Трехмерная Нотация

Запись ходов расширяет стандартную шахматную нотацию, добавляя координату слоя через двоеточие — `[Фигура][Исходная_клетка]:[Слой] — [Целевая_клетка]:[Слой]`.

*   `Ne2:1 — f3:4` — Конь прыгает с клетки $e2$ 1-го слоя на $f3$ 4-го слоя.
*   `e2:2 — e4:2` — Ход пешки в пределах своего слоя supporting авангарда.
*   `Bf1:1 x d3:3` — Слон с 1-го слоя берет вражескую фигуру на 3-м слое.

## 🧪 Тестирование (QA)

В проект интегрирован тестовый фреймворк `unittest`. Вы можете запустить тесты логики ядра (Backend) отдельно от графической оболочки с помощью команды:
```bash
python -m unittest spheres_of_power.py
```

Тесты проверяют:
1. Корректность инициализации стартового пула из 52 фигур.
2. Точность позиционирования и поиска объектов по координатам.
3. Математические ограничения трехмерного луча движения Ладьи.
4. Работу триггера «Friendly Fire» (запрет уничтожения собственных фигур).
